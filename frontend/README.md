# Frontend Prototype – Watch Party

Ce dossier contient un prototype de frontend développé individuellement avec Vue.js.

## Objectif
Mettre en place une première interface permettant :
- d’afficher les WatchParties
- de consulter les détails d’une WatchParty
- de simuler des interactions (changement d’état du match)

## État du projet
Ce frontend est un prototype non finalisé.
Il n’a pas été intégré au projet principal afin de ne pas impacter la stabilité de la version finale.

## Fonctionnalités implémentées
- Liste des WatchParties
- Navigation entre pages (routing)
- Page de détails d’une WatchParty
- Données simulées (mock API)

## Lancer le projet

```bash
npm install
npm run dev
